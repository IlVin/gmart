# GopherMart Entry Point

Данная директория содержит основной исполняемый файл приложения. Здесь происходит первичная настройка окружения и запуск жизненного цикла сервера.

## Ответственность

1.  **Environment Setup**: Загрузка переменных из `.env` файлов и автоматическая адаптация имен переменных под требования фреймворка Huma (через механизм `FixEnv`).
2.  **CLI Handling**: Использование `humacli` для обработки аргументов командной строки и автоматической генерации справки.
3.  **Infrastructure Bootstrap**: 
    - Инициализация реестра метрик Prometheus.
    - Создание пула соединений с PostgreSQL.
    - Запуск и контроль выполнения миграций базы данных.
4.  **Signal Handling**: Реализация **Graceful Shutdown** через `signal.NotifyContext` для перехвата сигналов завершения (SIGINT, SIGTERM).

## Особенности реализации

-   **FixEnv (Reflective Mapping)**: Функция использует рефлексию для автоматического маппинга полей структуры `CLIOptions` в переменные окружения с префиксом `SERVICE_`. Это позволяет приложению соответствовать ожиданиям `huma`, не заставляя пользователя вручную прописывать префиксы в конфигурации.
-   **Hook-based Startup**: Весь процесс запуска инкапсулирован в хуки `humacli`, что гарантирует правильный порядок инициализации и очистки ресурсов.
-   **Migration Safety**: Миграции запускаются с использованием `context.Background()`, чтобы гарантировать их завершение даже при получении сигнала остановки в момент старта, что предотвращает повреждение схемы БД.

## Аргументация паттернов

-   **Singleton-like Resources**: Инстансы базы данных и реестра метрик создаются один раз на уровне `main` и пробрасываются в `serve.Serve` (Dependency Injection), что исключает скрытые глобальные состояния.
-   **Defer-Based Cleanup**: Использование `defer` для закрытия соединений с БД гарантирует освобождение ресурсов при любом сценарии завершения функции `run`.
-   **Fail-Fast Strategy**: Приложение немедленно завершает работу с ненулевым кодом выхода, если критические компоненты (БД, миграции) не смогли инициализироваться.

## Запуск

```bash
# С использованием переменных окружения
DATABASE_URI=postgres://user:pass@localhost:5432/db ./gophermart

# С использованием флагов
./gophermart --database-uri="postgres://user:pass@localhost:5432/db"
```
