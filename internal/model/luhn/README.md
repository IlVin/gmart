# Luhn Algorithm Validator

Пакет `luhn` предоставляет инструменты для проверки строк на соответствие алгоритму Луна. В контексте проекта используется для валидации номеров заказов (Luhn check digit).

## Использование

```go
package main

import (
	"fmt"
	"gmart/internal/domain"
	"gmart/internal/model/luhn"
)

func main() {
	order := domain.OrderNumber("1234567812345670")
	if luhn.IsValid(order) {
		fmt.Println("Номер заказа валиден")
	}
}
```

## Особенности реализации

- **Любая длина**: Алгоритм корректно вычисляет контрольную сумму как для четной, так и для нечетной длины строки благодаря определению паритета (`parity`).
- **Валидация символов**: Функция возвращает `false`, если в строке встречается любой символ, отличный от цифры `0-9`.
- **Zero Allocation**: Реализация не выделяет дополнительной памяти в куче при проверке.

## Как это работает (кратко)

1. Каждая вторая цифра в последовательности удваивается.
2. Если результат удвоения больше 9, из него вычитается 9.
3. Все полученные цифры суммируются.
4. Результат должен быть кратен 10.
